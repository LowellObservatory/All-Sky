asc_logger module¶

Module used by AllSkyMediator, allsky, BrokerComm, camera and protoLoFits for logging and debug level logging.

The instance of asmlog is defined as the object ‘log_obj’ with a call to the lineno() method. The line number is used to assist with debugging and is followed with the body of the log entry both contained within an f-string literal for consistent formatting of log entries.
